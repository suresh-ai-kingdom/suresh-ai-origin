# Icon Placeholders

The extension requires three icon sizes:

- **icon16.png** (16×16px) - Toolbar icon (small)
- **icon48.png** (48×48px) - Extension management page
- **icon128.png** (128×128px) - Chrome Web Store listing

## Generate Icons

Use any of these tools to create icons:

1. **Canva** (https://canva.com)
   - Create 128×128 design
   - Export as PNG
   - Resize to 16×16 and 48×48

2. **Figma** (https://figma.com)
   - Design icon with ✨ symbol
   - Use purple gradient (#667eea → #764ba2)
   - Export at 1x, 3x, 8x scales

3. **Online Icon Generator**
   - https://www.favicon-generator.org/
   - Upload logo, generate all sizes

## Icon Design Guidelines

**Theme**: Purple gradient with sparkle (✨) symbol

**Colors**:
- Primary: #667eea (Purple)
- Secondary: #764ba2 (Dark purple)
- Accent: White/Gold

**Style**: Modern, minimal, flat design

**Symbol**: ✨ (sparkle/star representing "rare")

## Quick Setup

For testing, you can use emoji-based icons:

1. Go to https://favicon.io/emoji-favicons/sparkles/
2. Download sparkles favicon
3. Rename files:
   - favicon-16x16.png → icon16.png
   - favicon-32x32.png → icon48.png (resize to 48×48)
   - android-chrome-512x512.png → icon128.png (resize to 128×128)

Or use any image editor to create solid purple squares with ✨ emoji in the center.
