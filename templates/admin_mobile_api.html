Authentication & Tokens

Mobile clients authenticate using JWT. Tokens expire in 7 days.

POST /api/v2/auth/login
{
  "email": "user@example.com",
  "password": "password"
}

Response:
{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "user_id": "user_123",
  "tier": "pro"
}

Use token in Authorization header: Bearer {token}

API Endpoints

Authentication

POST /api/v2/auth/login

Authenticate with email/password, get JWT

POST /api/v2/auth/signup

Create new account

POST /api/v2/auth/refresh

Refresh expired token

Content Generation

GET /api/v2/content/prompts

List available AI prompts

POST /api/v2/content/generate

Generate content from prompt

GET /api/v2/content/{id}

Get generated content by ID

Offline Sync

POST /api/v2/sync/push

Push offline changes to server

GET /api/v2/sync/pull

Pull server changes

Analytics

GET /api/v2/analytics/usage

Get user's usage stats & credit balance
