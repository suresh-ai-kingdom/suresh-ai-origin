Web SDK - Quick Start

Install via npm:

npm install @suresh-ai/sdk

Basic usage:

import { SureshAIClient } from '@suresh-ai/sdk'; const client = new SureshAIClient({ baseURL: 'https://api.suresh-ai.com', apiVersion: 'v2' }); const { token } = await client.auth.login('user@example.com', 'password'); const content = await client.content.generate({ prompt: 'Write a blog post', templateId: 'blog_post' });

Features:

  • JWT Authentication
  • Offline-first with IndexedDB
  • Automatic sync
  • Full TypeScript support

iOS SDK - Quick Start

Install via CocoaPods:

pod 'SureshAISDK', '~> 2.0'

Basic usage:

import SureshAISDK let client = SureshAIClient( baseURL: "https://api.suresh-ai.com", apiVersion: "v2" ) let response = try await client.auth.login( email: "user@example.com", password: "password" )

Features:

  • Swift async/await
  • CoreData offline storage
  • Push notifications
  • Background sync

Android SDK - Quick Start

Add to build.gradle:

implementation 'com.suresh-ai:sdk:2.0.0'

Basic usage:

val client = SureshAIClient.Builder() .baseURL("https://api.suresh-ai.com") .apiVersion("v2") .build() lifecycleScope.launch { val response = client.auth.login("user@example.com", "password") }

Features:

  • Kotlin coroutines
  • Room Database persistence
  • WorkManager background sync
  • Push notifications

Python SDK - Quick Start

Install via pip:

pip install suresh-ai-sdk

Basic usage:

from suresh_ai import SureshAIClient client = SureshAIClient( base_url="https://api.suresh-ai.com", api_version="v2" ) token = client.auth.login("user@example.com", "password") content = client.content.generate( prompt="Write a blog post", template_id="blog_post" )

Features:

  • Async/await support
  • Type hints
  • Batch operations
  • Request retry logic
